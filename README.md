# Astitva Prana Yoga Shala

Single-page static website for a Mysuru-based yoga instructor with 10+ years of experience. The site is designed to be hosted on GitHub Pages.

## Preview

- Hero section with auto-sliding background images
- Sections for About, Offerings, Schedule, Testimonials, and Contact
- Clean typography and responsive layout via CSS

## Running Locally

1. Clone the repository
2. Open `index.html` in a browser

No build step is required.

## Deploying to GitHub Pages

1. Push the `index.html`, `styles.css`, and this `README.md` to the repository
2. In GitHub, go to Settings â†’ Pages and set the source to the root of the main branch

The site will be available at `https://<username>.github.io/<repo>/`.

## Updating Hero Images

Replace the placeholder URLs inside `.hero-slide-1`, `.hero-slide-2`, and `.hero-slide-3` in `styles.css` with your own image paths. Each slide runs on a 12-second loop with fade and zoom transitions.
